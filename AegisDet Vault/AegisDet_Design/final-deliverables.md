# Final Deliverables

## Required research artifacts

- public or reviewable code repository with reproducible setup;
- dataset card, license/source manifest, labeling rules, and split audit;
- frozen YOLO26n baseline checkpoint identity and ONNX export;
- AegisDet-Mini implementation with tested router, crop, remap, and merge;
- complete ablation matrix for data, crop, router, context, distillation, and quantization;
- raw and summarized latency results with p50/p90/p99;
- failure gallery and limitation analysis;
- runtime matrix for supported backends;
- final technical report, README, model card, and demo video;
- experiment configs, run metadata, and negative results.

## Definition of complete

Another technically capable person must be able to understand the claim, recreate the environment, identify the exact data/model/runtime used, rerun the main comparison, and see both strengths and limitations.
