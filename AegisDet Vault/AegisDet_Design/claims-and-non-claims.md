# Claims and Non-Claims

## Allowed after evidence

- AegisDet improved a named metric on the locked wildlife test set under a named backend.
- Selective refinement activated on a measured fraction of frames.
- A specific context block improved or failed to improve EdgeScore.
- An exported runtime achieved measured latency on exact hardware.

## Never claim without broader evidence

- universal state of the art;
- superior performance on all edge devices;
- invention of hybrid CNN-transformer detection;
- safety-critical reliability;
- energy efficiency without power measurement.
