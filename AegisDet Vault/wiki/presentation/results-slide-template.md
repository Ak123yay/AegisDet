# Results Slide Template

Use one claim per slide. Show the frozen baseline and final variant in a table with mAP50-95, small-object AP, precision, recall, p50/p90/p99 latency, refinement rate, and artifact size. Include hardware, runtime, input size, precision, and test-set size in the footer. Add a Pareto plot and one honest sentence stating whether the hypothesis was supported. Never show only the best qualitative image.
