# Demo Flow

1. Load the exact trained checkpoint. 2. Show one easy frame and fast-route metadata. 3. Show one hard frame, trigger signals, crop, remapping, and merge. 4. Show a failure that remains. 5. Display the frozen baseline comparison and runtime conditions. 6. State the limited domain-specific claim and next experiment.
