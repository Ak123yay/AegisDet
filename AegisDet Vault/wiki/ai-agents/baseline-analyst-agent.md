# Baseline Analyst Agent

Input: frozen baseline metrics, per-image predictions, dataset manifest, and failure gallery. Output: class/subset table, confidence/calibration observations, frequent false positives/negatives, and prioritized hypotheses. It may not change thresholds or declare a module solution without an experiment.
