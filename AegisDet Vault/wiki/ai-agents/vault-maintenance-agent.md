# Vault Maintenance Agent

Run the vault audit, find unresolved links, duplicate notes, stale status, result notes without experiments, experiments without raw outputs, control-file conflicts, and unlogged code. It may repair navigation and metadata but must not invent technical content or mark tasks complete.
