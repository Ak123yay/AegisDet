# Mentor Update Format

A mentor update should fit on one page: research question; current phase; work completed; one table or figure of real evidence; one failure or surprise; blocker; decision needed; and the next experiment. Never send only a list of coding activity. Distinguish results from plans and include exact links to the experiment note or report.
