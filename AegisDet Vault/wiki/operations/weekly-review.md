# Weekly Review

Once per week, audit the active phase rather than planning every future phase. Review unchecked tasks, recent code changes, experiment outputs, failing tests, dataset changes, and decisions. Confirm the validation/test split is still frozen. Archive abandoned approaches with reasons. Produce a short report containing work completed, evidence, blockers, highest-risk assumption, and the next seven-day objective.
