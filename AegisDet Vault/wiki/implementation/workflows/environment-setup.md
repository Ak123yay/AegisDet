# Environment Setup

## Windows / RTX 3060 Ti

1. Install Python 3.11.
2. Create and activate `.venv` inside `../aegisdet/`.
3. Install a CUDA-compatible PyTorch build using the command generated by the official PyTorch installer page.
4. Install `requirements.txt` and editable package.
5. Run `scripts/check_environment.py`.
6. Save the JSON output in `output/environment-check.txt`.
7. Run unit tests before training.

## Gate

Phase 0 does not pass until CUDA is visible, Ultralytics imports, `yolo26n.pt` loads, and the deterministic tests pass.
