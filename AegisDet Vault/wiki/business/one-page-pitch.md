# One Page Pitch

**Problem:** nano detectors are fast but often miss tiny or obscured wildlife; full high-resolution or sliced inference is costly. **Method:** use a fine-tuned YOLO26n fast path and route only difficult detections to bounded local refinement, then test a tiny global-context addition. **Evidence plan:** frozen split, same backend, small-object metrics, latency percentiles, failure analysis, and runtime export. **Deliverable:** open, reproducible code and report with honest limitations.
